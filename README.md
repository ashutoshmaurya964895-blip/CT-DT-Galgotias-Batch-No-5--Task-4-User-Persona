# CT&DT Galgotias - Batch No: 5
## Task#04 – User Persona (Techniques of Empathy Building)

## 📋 Project Objective
This repository tracks our target customer empathy matrix designed for Task 4. Using structured user profile research within the empathy building framework, this persona document maps the core demographic dimensions, behavior dynamics, psychological traits, and user journey gaps necessary to ground our software design choices in authentic user needs.

## ⚙️ Empathy Matrix Dimensions
Our interactive user persona canvas explores the following strategic customer segments:
* **Demographic Information:** Establishing baseline contextual user identities, backgrounds, and professional profiles.
* **Goals & Motivations:** Pinpointing measurable performance outcomes and lifestyle needs driving user adoption.
* **Psychographics & Preferences:** Outlining day-to-day behavioral traits, platform preferences, and interaction mental models.
* **Friction Points & Challenges:** Mapping out operational frustrations and design gaps currently limiting user success.
